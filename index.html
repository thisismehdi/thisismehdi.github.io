  • vCenter-Inventory-Manager

    Java/Angular

    Inventaire vCenter is a web-based application designed to simplify the management and monitoring of virtual servers (VMs). It integrates seamlessly with VMware vCenter via a custom RESTful API, enabling real-time data synchronization and VM tracking. The system includes a secure authentication mechanism powered by Azure AD and MSAL, along with real-time notifications for VM modifications. Users can generate detailed CSV reports on resource consumption and use a calendar-based filtering feature for better data visualization. Developed using Angular and Spring Boot, the project follows Agile methodologies, with JIRA used for efficient task tracking and collaboration.

    Key Features :

    RESTful API Integration,Secure Authentication System, Data Export Functionality, Real-Time VM Monitoring, Notifications and Alerts, User-Friendly Inventory Management
